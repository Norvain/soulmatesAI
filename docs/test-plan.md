## 附录：接口错误码参考

| HTTP 状态码 | 含义 | 常见场景 |
|------------|------|---------|
| 400 | 请求参数错误 | 空内容提交、缺少必填字段、评论用户动态 |
| 401 | 未授权 | Token 过期或缺失 |
| 404 | 资源不存在 | 对话/角色/瞬间不存在或非当前用户所有 |
| 409 | 冲突 | 重复开始已完成的关系事件（非 replay 模式） |
| 500 | 服务器内部错误 | AI 生成失败、数据库异常 |
